# Fallback Avatar

A component which automatically sets the avatar as a fallback avatar on upload.
Attach it to the avatar's root object (the object with VRC Avatar Descriptor).

A fallback avatar is the avatar shown to players who don't allow your avatar to be displayed.
Only avatars with a performance rank of Good or better can be set as fallback avatars.

## Usage

1. Attach this component to the avatar's root object.
2. Upload the avatar for Mobile (Android or iOS).
3. When the performance rank is Good or better, the avatar is automatically made available as a fallback avatar after the upload.

There are no settings.
