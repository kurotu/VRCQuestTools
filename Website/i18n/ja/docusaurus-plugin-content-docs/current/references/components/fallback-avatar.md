# Fallback Avatar

このコンポーネントは、アバターがパフォーマンス要件を満たしている場合、アップロード後に自動的にフォールバックアバターとして設定します。

## 動作原理

このコンポーネントを付けたアバターをアップロードすると：

1. アバターがビルドされ、VRChatにアップロードされます
2. アップロードしたプラットフォームのパフォーマンスランクが計算されます
3. パフォーマンスランクがGood以上の場合、自動的にフォールバックアバターとして設定されます
4. パフォーマンスランクがGood未満の場合、警告がログに記録されますがアップロードは続行されます

## 要件

アバターをフォールバックアバターとして設定するには、VRChatのパフォーマンス要件を満たす必要があります：

- アップロードするプラットフォームでパフォーマンスランクが **Good** または **Excellent** である必要があります
- フォールバック機能を完全に活用するには、PCとAndroidの両方のバージョンがこれらの要件を満たすことが理想的です

:::info
VRChatの[フォールバックアバターシステム](https://docs.vrchat.com/docs/avatar-fallback-system)によると、フォールバックアバターは他のユーザーのアバターが読み込めない場合やブロックされている場合に使用されます。アバターをフォールバックとして設定することで、他のユーザーのエクスペリエンスを向上させることができます。
:::

## 使い方

1. アバターのルートオブジェクト（VRC Avatar Descriptorがあるゲームオブジェクト）に `VQT Fallback Avatar` コンポーネントを追加します
2. VRCQuestTools Avatar BuilderまたはVRCSDKコントロールパネルを使用してアバターをアップロードします
3. パフォーマンス要件を満たしている場合、コンポーネントが自動的にフォールバックとして設定を試みます

## 注意事項

- これは設定可能なプロパティを持たないマーカーコンポーネントです
- このコンポーネントは実行時にアバターに影響を与えません - ビルドプロセス中に削除されます
- VRCQuestTools Avatar Builderウィンドウのトグルを使用して、手動でフォールバック設定を制御することもできます
- アバターがパフォーマンス要件を満たしていない場合、フォールバックとして設定されませんが、アップロードは成功します
