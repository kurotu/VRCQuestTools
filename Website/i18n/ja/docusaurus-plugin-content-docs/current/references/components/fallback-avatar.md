# VQT Fallback Avatar

アップロード後、PCとAndroidの両方がパフォーマンス要件を満たしている場合、このアバターを自動的にフォールバックアバターとして設定します。

## フォールバックアバターとは？

VRChatのフォールバックアバターシステムを使用すると、アバターをフォールバックオプションとして指定できます。ユーザーの安全設定によってあなたのアバターが非表示になっている場合、一般的なロボットアバターの代わりにフォールバックアバターが表示されます。これにより、あなたと他のユーザーの両方にとってより良い体験が提供されます。

アバターをフォールバックとして使用するには、VRChatのパフォーマンス要件を満たす必要があります：
- PCとAndroidの両方のバージョンが**Good**または**Excellent**の評価である必要があります
- 両方のプラットフォーム向けにアバターをアップロードする必要があります

## 使い方

1. アバターのGameObjectのルートに**VQT Fallback Avatar**コンポーネントを追加します
2. PCとAndroidの両方のプラットフォーム向けにアバターをアップロードします
3. 両方のバージョンがパフォーマンス要件（Good以上）を満たしている場合、VRCQuestToolsはアップロード後に自動的にアバターをフォールバックアバターとして設定します

:::tip
アバターがパフォーマンス要件を満たしていることを確認するには、アップロードする前にVRChat SDK Control Panelでパフォーマンス評価を確認してください。
:::

:::info
このコンポーネントはプロパティを持たないマーカーコンポーネントです。アップロード後にこのアバターをフォールバックとして設定する必要があることをVRCQuestToolsに通知するだけです。
:::

## 動作

- コンポーネントは各アップロード後にパフォーマンス評価をチェックします
- パフォーマンス要件が満たされていない場合、Unityコンソールに警告が記録されます
- コンポーネントはAndroid/iOSビルドのみを処理します - PCのみのアップロードはスキップされます
- アバターがすでにフォールバックとして設定されている場合、変更は行われません

## 要件

- VRChat SDK3 - Avatars
- PCとAndroidの両方のアバターバージョンがアップロードされていること
- 両方のプラットフォームで**Good**または**Excellent**のパフォーマンス評価
