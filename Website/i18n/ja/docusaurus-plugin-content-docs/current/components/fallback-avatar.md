# Fallback Avatar

アップロード時に、アバターを自動的にフォールバックアバターとして設定するコンポーネントです。
アバターのルートオブジェクト（VRC Avatar Descriptor のあるオブジェクト）に追加します。

フォールバックアバターは、アバターの表示を許可していないプレイヤーからの見た目として使われるアバターです。
フォールバックアバターとして設定できるのは、パフォーマンスランクが Good 以上のアバターだけです。

## 使い方

1. アバターのルートオブジェクトにこのコンポーネントを追加します。
2. Mobile (Android または iOS) 向けにアバターをアップロードします。
3. パフォーマンスランクが Good 以上であれば、アップロード後に自動でフォールバックアバターとして使用可能に設定されます。

設定項目はありません。
