# Mesh Flipper

メッシュのポリゴンを複製して両面にする、または面を反転するコンポーネントです。
Mobile 用シェーダーでは裏面が描画されないため、スカートの裏地などが見えなくなることがあります。
このコンポーネントをメッシュのある GameObject に追加すると、ビルド時に両面化したメッシュを生成して差し替えます。
動作には NDMF が必要です。

![VQT Mesh Flipper の Inspector](/img/mesh-flipper.png)

## 設定項目

| 項目 | 説明 |
|---|---|
| メッシュの向き | 「両面」はポリゴンを複製して両面にします。「反転」は面の向きを裏返します。 |
| PCで有効 | PC ビルドで処理を行います。 |
| Mobileで有効 | Mobile ビルドで処理を行います。 |
| マスクを使用する | マスクテクスチャで処理する範囲を限定します。 |
| マスクテクスチャ | 処理範囲を決めるテクスチャです。読み取り可能 (Read/Write) に設定されている必要があります。 |
| マスクモード | 「Flip White」はマスクの白い部分を、「Flip Black」は黒い部分を処理の対象にします。 |
| NDMFフェーズ | ポリゴン削減ツールの前後どちらで処理するかを選びます。 |

## 補足

- 両面化するとメッシュのポリゴン数が増えます。パフォーマンスランクに注意してください。
- PC ビルドで使用する場合、シェーダーの両面描画設定で同じ見た目を実現できることがあります。
- Avatar Optimizer などのポリゴン削減ツールと併用する場合は、「NDMFフェーズ」で処理順を調整できます。
