# Menu Icon Resizer

Expressions Menu のアイコンをビルド時にリサイズするコンポーネントです。
アイコンのテクスチャ容量を減らして、ダウンロードサイズを削減できます。
アバターのルートオブジェクトに追加します。
動作には NDMF が必要です。

:::info スクリーンショット準備中
ここに VQT Menu Icon Resizer の Inspector のスクリーンショットが入ります。
:::

## 設定項目

| 項目 | 説明 |
|---|---|
| サイズ変更モード (PC) | PC ビルドでのアイコンの扱いです。 |
| サイズ変更モード (Mobile) | Mobile ビルドでのアイコンの扱いです。 |
| 未圧縮のテクスチャを圧縮 | アイコンが圧縮されていなければ圧縮します。 |
| 圧縮形式 | Android 用のテクスチャ圧縮形式です。 |

サイズ変更モードは次から選びます。

- **Do Not Resize**：リサイズしません。
- **Remove**：アイコンを削除します。
- **Max 64x64**：64×64 ピクセルに収まるように縮小します。
- **Max 128x128**：128×128 ピクセルに収まるように縮小します。

## 補足

[VQT Avatar Converter Settings](./avatar-converter-settings.md) の「メニューアイコン設定」でも同じ処理ができます。
アバター変換を使わずにアイコンだけ処理したい場合に、このコンポーネントを使用します。
